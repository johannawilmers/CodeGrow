## How to run the app
cd codegrow
npm run dev

## Local environment setup
Copy `.env.example` to `.env.local` and fill in your Firebase values plus the admin password before running the app.

If you use Firebase Functions locally, copy `functions/.env.example` to a local `.env` file inside `functions/` and keep the secret values out of version control.
